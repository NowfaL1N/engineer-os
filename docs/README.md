# Documentation

This folder contains project documentation, guides, and specifications.

## Structure

- `api/` - API documentation and specifications
- `architecture/` - Architecture decisions and diagrams
- `guides/` - Development guides and tutorials
- `specs/` - Feature specifications

## Contributing

When adding new features, please document:
- API contracts in `api/`
- Architecture decisions in `architecture/`
- Usage guides in `guides/`
- Feature specs in `specs/`
